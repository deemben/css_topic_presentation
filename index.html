CSS Topic - Background Image

Add a Basic Background

In your css, simply add the path to your background. It can either be saved in your repository or be a web link.

You should also choose a background-color.

Your background image will be working at this point, but will not look good. It's time to customize it!

See the Pen Background Image by Benjamin (@deem-) on CodePen.

Customize & Beautify your Background

There is a large variety of tools for customizing your background image to make it look as good as possible.

  • background-image

    Changes your background to an image. By default, it adds the image in its original resolution, repeating to fill the screen if necessary.

  • background-position

    By default, the background locks to the top-left of your screen. There are three ways to change position: Length value (100px 5px), percentages (100% 5%), or keywords (top right)

  • background-size

    There are two keywords that can be used: cover and contain. Cover tells the browser to make sure the image always covers the entire screen, even if it has to stretch or crop the image. Contain will always show the entire image, even if it leaves blank space on the sides or bottom.

  • background-repeat

    By default, the background image repeats if it is smaller than the browser window. You can disable or alter that repetition with this property. repeat-x and repeat-y control if the image repeats along those axis. no-repeat disables image repetition. space repeats the image in both directions with even spaces to avoid cropping, and round will stretch or squish the images to fill available space.

  • background-attachment

    This specifies how the background moves when scrolling down the web page. Scroll is the default value. The image scrolls along with the main view in a fixed position. Fixed stays in a fixed location no matter what. It acts like a real window, staying in place no matter how your perspective changes. Local scrolls both with the main view and the local view.

  • background-clip

    background-clip lets you control how far a background image or color extends beyond an element's padding or content. There are four values which each change the appearance. border-box is the default value. This allows the background to extend all the way to the outside edge of the element's border. padding-box clips the background at the outside edge of the element's padding and does not let it extend into the border. content-box clips the background at the edge of the content box. inherit applies the background-clip setting of the parent to the selected element.

  • background-origin

    This defines where the background is painted. This has no effect if the background is fixed. The values are the same as the four for background-clip, except it resizes the background rather than clipping it.

  • background-color

    The background-color property simply applies solid colors as background on an element. This can be done by simply writing the name of the color, or by using a code (Hex, RGB, RGBa, HSL, or HSLa).
